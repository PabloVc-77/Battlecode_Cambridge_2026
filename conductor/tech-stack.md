# Tech Stack: Battlecode Cambridge 2026 Bot

## Language
- **Python 3**: Primary language for bot development, chosen for its flexibility and the richness of its libraries for algorithmic development.

## Frameworks & SDKs
- **Battlecode 2026 Python SDK**: The official SDK provided for the competition, managing the game engine interactions, unit commands, and map data.

## Environment & Data
- **File-based System**: Using local files (e.g., cambc.toml, .map26, .replay26) to manage project configuration, map data, and match replays.

## Tooling
- **Git**: For version control and collaborative development across different bot iterations.
